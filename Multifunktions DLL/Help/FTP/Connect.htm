Function FTP.Connect

Baut eine Verbindung zu einem FTP-Server auf

Deklaration

Public Function Connect(IEopen As IE_OpenType, _

                                            ByVal Server As String, _

                                            Optional UserName As String = "anonymous", _

                                            Optional Password As String = "guest", _

                                            Optional Port As Long = 21, _

                                            Optional PassiveConnection As Boolean = True) _

                                                 As Long

Argumente

]IEopen]

INTERNET_OPEN_TYPE_PRECONFIG

    Verbindung wird aus den Windows-Einstellungen übernommen

INTERNET_OPEN_TYPE_DIRECT

    Direkte Verbindung

INTERNET_OPEN_TYPE_PROXY

    Verbindung geht über einen Proxy

INTERNET_OPEN_TYPE_PRECONFIG_WITH_NO_AUTOPROXY

    Verbindung wird aus den Windows-Einstellungen übernommen,

    ohne Proxy

Server

Der Server, zu dem die Verbindung aufgebaut werden soll

UserName

Optional

Mögliche Angabe eines Benutzers (Vorgabe ist „anonymous“)

Password

Optional

Mögliche Angabe eines Passwotes (Vorgabe ist „guest“)

Port

Optional

Mögliche Angabe des Portes, über den die FTP-Verbindung gehen soll (Vorgabe ist 21)

PassiveConnection

Optional

Wird True übergeben, so wird eine passive Verbindung aufgebaut,

bei False eine aktive.

Rückgabewert

-1

Eine Verbindung zu diesem Server besteht bereits

0

Verbindung war nicht erfolgreich

1

Die Verbindung konnte erfolgreich hergestellt werden

Sonstige Hinweise

Ist eine Verbindung erfolgreich, kann nun der Servername (in dem Beispiel „xx.xxx.xx.xxx“)

in anderen Funktionen übergeben werden

 

Beispiel

    Dim FTP As New WindowAPI.FTP

 

    ‘Eine anonyme Verbindung aufbauen

    MsgBox FTP.Connect(INTERNET_OPEN_TYPE_PRECONFIG, _

                                          "MeinServer")
