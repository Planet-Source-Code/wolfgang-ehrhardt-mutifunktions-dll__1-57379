Function Forms.tFade

Fadet die Transparenz einer Form

Deklaration

Public Function tFade(Form As Object, _

                                  ByVal FadeStart As Byte, _

                                  ByVal FadeStop As Byte, _

                                  Optional StartAtCurrentTransparenz As Boolean = False, _

                                  Optional FadeStep As Integer = 5, _

                                  Optional Speed As Integer = 10) _

                                      As Boolean

Argumente

Form

Form, deren Transparenz gefadet werden soll

FadeStart

Starttransparenz

FadeStop

Endtransparenz

StartAtCurrentTransparenz

Optional

Wird True übergeben, so wird die Starttransparenz auf die aktuelle Transparenz gesetzt

FadeStep

Optional

Entscheidet in welchen Intervall gefadet wird

Speed

Optional

Entscheidet wie schnell gefadet wird

Rückgabewert

True wenn die Form die Endtransparenz erreicht hat

 

Beispiel

'1 x Form 'Form1

 

Private Sub Form_Load()

    Dim FRMS As New WindowAPI.Forms

   

    Me.Show

   

    MsgBox FRMS.tFade(Me, 0, 0, True)

   

    Unload Me

   

End Sub
