Function System.SystemDirectory

Gibt verscheiedene System-Verszeichnise wieder

Deklaration

Public Function SystemDirectory(SysInfo As SystemDirNFO) As String

Argumente

sdCommonProgramFilesDir (1)

Gibt das Verzeichnis für „Gemeinsame Dateien“ wieder

sdAllUsersProfileDir (2)

Gibt das Verzeichnis von „All Users“ wieder

sdUserProfileDir (4)

Gibt das Verzeichnis des angemeldeten Users zurück

sdWindowsDirectory (5)

Gibt das Windows-Verzeichnis zurück

sdWindowsSystemDir (6)

Gibt das Windows-System-Verzeichnis zurück

sdWindowsTempDir (7)

Gibt das Temporär-Verzeichnis wieder

sdStartmenuDir (8)

Gibt das Verzeichnis des Startmenus wieder

sdDesktopDirectory (9)

Gibt das Verzeichnis des Desktops wieder

sdProgramFilesDir (10)

Gibt das Verzeichnis von „Programmen“ wieder

sdPersonalDirectory

Gibt das Verzeichnis von „Eigenen Dateien“ wieder

sdFavoritDirectory (12)

Gibt das Verzeichnis der Internet Favoriten wieder

sdAutoStartDirectory (13)

Gibt das AutoStart-Verzeichnis wieder

sdRecentDocumentDir (14)

Gibt den Dokumentenordner „Recent“ wieder

sdSendToDirectory (15)

Gibt das Verzeichnis von „Senden an“ wieder

sdNewShellDir (16)

Gibt das Verzeichnis für Vorlagen wieder

sdNetworkDirectory (17)

Gibt das Verzeichnis für „Netzwerkumgebung“ wieder

sdFontDirectory (18)

Gibt das Verzeichnis der Schriftarten wieder

sdTemporaryInternetFilesDir (19)

Gibt das Verzeichnis der Temporären Internetdateien wieder

sdUserDesktopDir (20)

Gibt das Verzeichnis des Desktop-Verzeichnises des angemeldeten Users wieder

sdUserOwnMusicDir (21)

Gibt das Verzeichnis für „Eigene Musik“ wieder

sdUserOwnPicsDir (22)

Gibt das Verzeichnis für „Eigene Bilder“ wieder

sdUserCookiesDir (23)

Gibt das Cookie-Verzeichnis des angemeldeten Users wieder

sdAllUserDesktopDir (24)

Gibt das Desktop-Verzeichnis für All Users wieder

sdUserAppDataDir (25)

Gibt das Verzeichnis der Anwendungsdaten des angemeldeten Users wieder

sdUserQuickLaunchDir (26)

Gibt das Quick-Launch-Verzeichnis des angemeldeten Users wieder

Rückgabewert

Das gewünschte System-Verzeichnis

 

Beispiel

    Dim SYS As New WindowAPI.System

   

    MsgBox SYS.SystemDirectory(sdWindowsDirectory)

    MsgBox SYS.SystemDirectory(sdStartmenuDir)
