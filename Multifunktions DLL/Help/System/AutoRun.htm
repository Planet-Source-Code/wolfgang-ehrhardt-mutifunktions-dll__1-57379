Function System.AutoRun

Trägt einen Eintrag in der Registry ein, damit ein Programm bei jedem Windowsstart mitgestartet wird

Deklaration

Public Function AutoRun(ByVal Field As String, _

                                             ByVal EXEpath As String) _

                                                 As Boolean

Argumente

Field

Beschreibung des zu schreibenden Feldes

EXEpath

Pfad zu der Anwendung, die gestartet werden soll

Rückgabewert

True wenn der Wert erfolgreich in die Registry geschrieben wurde

 

Beispiel

    Dim SYS As New WindowAPI.System

   

    MsgBox SYS.AutoRun("Notepad", "C:\Windows\System32\Notepad.exe")
