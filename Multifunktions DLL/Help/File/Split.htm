Function File.Split

Teilt (splittet) eine Datei in mehrere Datei auf

Deklaration

Public Function Split(ByVal File As String, _

                                      ByVal KBperFile As Long, _

                                      Optional OutputFile As String, _

                                      Optional CreateBatch As Boolean = False) _

                                          As Long

Argumente

File

Datei, die gesplittet werden soll

KBperFile

Größenangabe der gesplitteten Dateien in KiloByte

OutputFile

Optional

Mögliche Angabe einer Dateivorgabe

Wird OutputFile nicht übergeben, so wird eine Datei anhand der Quell-Datei erstellt

CreateBatch

Optional

Wird True übergeben, so wird eine Batch-Datei erstellt, die die Dateien wieder zusammenfügt

Rückgabewert

-3

Batch-Datei konnte nicht erstellt werden

-2

Quelldatei ist kleiner als die angegebenen gesplitteten Dateien

-1

Quell-Datei existiert nicht

<>

Sonstige Fehlercode

Interne Fehlernummer

 

Beispiel

    Dim FILE As New WindowAPI.FILE

   

    MsgBox FILE.Split("C:\Windows\Explorer.exe", 50, "C:\SplitTest", True)
