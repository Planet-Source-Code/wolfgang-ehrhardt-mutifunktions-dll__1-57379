Function Dektop.WriteOnDesktop

Macht es möglich eine Schrift auf den Desktop erscheinen zu laßen

Deklaration

Public Function WriteOnDesktop(ByVal Text As String, _

                                           ByVal Top As Long, Left As Long, _

                                           Optional Center As Boolean, _

                                           Optional StayOnTop As Boolean = True, _

                                           Optional FontName As String = "Times New Roman", _

                                           Optional FontSize As Long = 36, _

                                           Optional FontColor As Long = vbRed, _

                                           Optional FontBold As Boolean = False, _

                                           Optional FontItalic As Boolean = False, _

                                           Optional FontUnderline As Boolean = False, _

                                           Optional FontStrikethrough As Boolean = False, _

                                           Optional Clear As Boolean = False) _

                                                 As Boolean

Argumente

Text

Der Text, der auf dem Desktop erscheinen soll

Top

Ausrichtung zur oberen Monitorgrenze

Center

Optional

Wird True übergeben, so wird die Schrift in der Mitte des Bildschirmes dargstellt

StayOnTop

Optional

Wird True übergeben, so ist die Schrift über allen anderen Dialogen

FontName

Optional

Schriftart, in der die Schrift dargestellt werden soll

FontSize

Optional

Größe der Schrift

FontColor

Optional

Bestimmt die Farbe der Schrift

FontBold

Optional

Bestimmt die Schrift fett dargestellt werden soll

FontItalic

Optional

Gibt an, ob der Text kursiv dargestellt werden soll

FontUnderline

Optional

Bestimmt, ob der Text unterstrichen ist

FontStrikethrough

Optional

Bestimmt, ob der Text durchstrichen ist

Clear

Optional

Bestimmt, das der Text wierder entladen wird

Rückgabewert

True wenn die Schrift ordnungsgemäß auf dem Desktop angezeigt wird

 

Beispiel

    Dim DESK As New WindowAPI.Desktop

   

    MsgBox DESK.WriteOnDesktop("Hello World", 0, 0, True, , , , , True)
