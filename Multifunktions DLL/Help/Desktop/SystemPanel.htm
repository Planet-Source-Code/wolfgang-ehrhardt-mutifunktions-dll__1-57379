Function Desktop.SystemPanel

Öffnete verschiedene Einträge aus der Systemsteuerung

Deklaration

Public Function SystemPanel(Run As PanelOptionNFO) As Boolean

Argumente

[Run]

poControlPanel

    Öffnet die Systermsteuereung

poRemove_Software

   Öffnet Software entfernen

poAdd_Software

     Öffnet Software hinzufügen

poAdd_Remove_WindowsComponents

    Öffnet Windows Komponenten Hinzufügen/entfernen

poSystemInformation

     Öffnet Systeminformation

poComputerNameSettings

    Öffnet Coputernamensettings

poHardwareManager

    Ögffnet den Hardwaremanager

poTime_Date_Settings

     Öffnet Zeit/Datum Einstellungen

poRegional_Settings

    Öffnet regionale Einstellungen

poDisplay_Settings

    Öffnet Bildschirmeinstelungen

poInternet_Settings

    Öffnet Internet Einstellungen

poKeyboard_Settings

    Öffnet Tastatur Einstellungen

poMouse_Settings

    Öffnet Mouse Einstellungen

poModem_Settings

    Offnet Modem Einstellungen

poSound_Settings

    Öffnet Multimedia Einstellungen

poNew_Printer

     Öffnet Dialog für einen neunen Drucker

poInput_Assistance

    Öffnet den Hilfseingavbedialog

poCountrie_Attitudes

    Öffnet die Ländereinstellungen

poODCB_Settings

    Öffnet Datenbankeinstellungen

poGame_Settings

    Öffnet die Gameconreoller

poWindows_Update

    Öffnet das Windows-Update

Rückgabewert

True wenn Dialog ordnungsgemaß angezeigt wird

 

Beispiel

    Dim Desk As New Desktop

   

    MsgBox Desk.SystemPanel(poAdd_Software)

    MsgBox Desk.SystemPanel(poHardwareManager)
