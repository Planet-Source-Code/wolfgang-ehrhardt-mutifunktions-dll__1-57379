Function Grafic.vListImage

Ein virtuelles ListImage-Controll

Deklaration

Public Function vListImage(Command As vImageListCommand, _

                                                 Index As Long, _

                                                 Optional Picture As StdPicture, _

                                                 Optional Dimension As vImageListNfo) _

                                                     As StdPicture

Argumente

[Command]

viADD

    Fügt ein Bild hinzu

    In Index ist nun der Index des neuen Bildes

viGET

    Ruft ein Bild ab

    Index (benötigt) ist der Index des Bildes

    Rückgabe ist das abgerufene Bild

viDELETE

    Löscht ein Bild

    Index (benötigt) ist der Index des zu löschenden

    Bildes

viCLEAR

    Löscht alle Bilder aus dem Speicher

viITEMCOUNT

    Gibt die Anzahl der gespeicherten Bilder zurück

    Index enthält nun die Anzahl der gespeicherten Bilder

Index

Hängt von [Command] ab

Picture

Optional

Muß nur übergeben werden wenn [Command] = viADD ist und enthält das Bild, welches hinzugefügt werden soll

[Dimension]

Optional

Muß übergeben werden wenn [Command] = viADD ist und enthält die Information in welcher Dimension das Bild abgespeichert werden soll

 

vi16x16

    Das Bild wird in 16x16 abgespeichert

vi24x24

    Das Bild wird in 24x24 abgespeichert

vi32x32

    Das Bild wird in 32x32 abgespeichert

vi48x48

    Das Bild wird in 48x48 abgespeichert

viBYPICTURE

    Das Bild bleibt unverändert

Rückgabewert

Index

Hängt von [Command] ab

 

viADD

    Index enthält den Index des neu hinzugefügten Bildes

viITEMCOUNT

    Index enthält die Anzahl der gespeicherten Bilder

Ist [Command] = viADD, so wird das Bild mit dem Index [Index] zurückgegeben

 

Beispiel

'1 x Form 'Form1

'1 x PictureBox 'Picture1 (mit Bild)

'1 x PictureBox 'Picture2

 

Private Sub Form_Load()

    Dim I As Long

    Dim Gr As New Grafic

   

    Call Gr.vListImage(viADD, I, Picture1.Picture, vi32x32)

   

    MsgBox "Das hinzugefügte Bild hast den Index " & I

   

    Picture2 = Gr.vListImage(viGET, I)

   

    Call Gr.vListImage(viITEMCOUNT, I)

   

    MsgBox "Es befinden sich " & I & " Bild(er) im vListImage"

   

    Call Gr.vListImage(viCLEAR, 0)

   

End Sub
