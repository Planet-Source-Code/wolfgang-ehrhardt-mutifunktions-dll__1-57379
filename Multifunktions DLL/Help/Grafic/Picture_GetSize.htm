Function Grafic.Picture_GetSize

Gibt Höhe und Breite einer Bilddatei in Pixel wieder

Deklaration

Public Function Picture_GetSize(ByVal File As String, _

                                                         Width As Long, _

                                                         Height As Long) As Long

Argumente

File

Bilddatei, von der die Höhe und Breite wiedergegeben werden soll

Width

Wird zur Rückgabe benötigt

Height

Wird zur Rückgabe benötigt

Rückgabewert

-1

Quelldatei existier nicht

-2

Ungültige Bilddatei

1

Funktion erfolgreich

Width

Breite des Bildes

Height

Höhe des Bildes

 

Beispiel

'1 x Form 'Form1

'1 x Bilddatei 'C:\Test.bmp

 

Private Sub Form_Load()

    Dim H As Long, W As Long, Result As Long

    Dim Gr As New Grafic

   

    Result = Gr.Picture_GetSize("C:\Test.bmp", W, H)

   

    Select Case Result

        Case -2: MsgBox "Ungültiges Bild"

        Case -1: MsgBox "Quelldatei existiert nicht"

        Case 1: MsgBox "Höhe:  " & H & vbCrLf & _

                                     "Breite: " & W

    End Select

   

End Sub
