Function Misc.DateIs

Gibt zurück ob sich eine Datums-/Zeitangabe nur um ein Datum, nur um eine Zeitangabe oder beides handelt

Deklaration

Public Function DateIs(ByVal Dat As Date) As DateType

Argumente

Dat

Datums-/Zeitangabe, die überprüft werden soll

Rückgabewert

[DateType[

dtTIMEONLY (0)

    Es hanldet sich nur um eine Zeitangabe

dtDATEONLY (1)

    Es handelt sich nur um eine Datumsangabe

dtTIMEDATE (2)

    Es hanedelt dich um eine Datums- & Zeitangabe

dtUNKOWN (-1)

    Unbekanntes Format

    (sollte nicht vorkommen)

 

Beispiel

    Dim MISC As New WindowAPI.MISC

   

    MsgBox MISC.DateIs(Now)
