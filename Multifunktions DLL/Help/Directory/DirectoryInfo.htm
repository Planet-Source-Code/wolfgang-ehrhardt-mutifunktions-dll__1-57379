Sub Directory.DirectoryInfo

Gibt Größe, Dateienanzahl und Unterverzeichnisanzahl eines Verzeichnis wieder

Deklaration

Public Sub DirectoryInfo(ByVal Directory As String, _

                                            DirectorySize As Variant, _

                                            FileCount As Long, _

                                            DirectoryCount As Long, _

                                            Optional InclSubDirectorys As Boolean = False)

Argumente

Directory

Verzeichnis, aus dem die Informationen gelesen werden sollen

DirectorySize

Wird zur Rückgabe bebötigt

FileCount

Wird zur Rückgabe bebötigt

DirectoryCount

Wird zur Rückgabe bebötigt

InclSubDirectorys

Optional

Wird True übergeben, so werden die Unterverzeichnise mit einbezogen

Rückgabewert

DirectorySize

Größe des Verzeichnis in Bytes

FileCount

Anzanhl der Dateien

DirectoryCount

Anzahl der UNterverzeichnise

 

Beispiel

    Dim DI As New WindowAPI.Directory

    Dim S As Variant, C As Long, D As Long

   

    Call DI.DirectoryInfo("C:\", S, C, D, True)
