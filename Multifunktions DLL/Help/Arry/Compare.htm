Function Arry.Compare

Vergleicht 2 Arrays miteinander, ob diese identisch sind

Deklaration

Public Function Compare(sArray1 As Variant, _

                                             sArray2 As Variant) As Boolean

Argumente

sArray1

Array, das mit sArray2 verglichen wird

sArray2

Array, das mit sArray1 verglichen wird

Rückgabewert

True wenn sArray1 und sArray2 identisch sind

 

Beispiel

    Dim A As Variant, B As Variant

    Dim Ar As New WindowAPI.Arry

   

    ReDim A(0)

    ReDim B(0)

   

    A(0) = "Hello"

    B(0) = "Hello"

   

    MsgBox Ar.Compare(A, B)
