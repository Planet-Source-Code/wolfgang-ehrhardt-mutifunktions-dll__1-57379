Function Arry.Copy

Kopiert den Inhalt eines Arrays in ein anderes Array

Deklaration

Public Function Copy(sArray1 As Variant, _

                                       sArray2 As Variant) As Boolean

Argumente

sArray1

Array, das in sArray2 kopiert werden soll

sArray2

Array, in das sArray1 kopiert werden soll

Rückgabewert

True wenn sArray1 erfolgreich in sArray2 kopiert wurde

 

Beispiel

    Dim A As Variant, B As Variant

    Dim Ar As New WindowAPI.Arry

   

    ReDim A(1)

   

    A(0) = "Hello"

    A(1) = "World"

   

    MsgBox Ar.Copy(A, B)
