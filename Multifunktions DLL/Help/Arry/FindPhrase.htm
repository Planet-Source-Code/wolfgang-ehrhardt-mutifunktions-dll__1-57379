Function Arry.FindPhrase

Durchsucht ein Array nach einer bestimmten Phrase

Deklaration

Public Function FindPhrase(sArray As Variant, _

                                                 ByVal Find As String, _

                                                 Optional StartIndex As Long = -1, _

                                                 Optional EndIndex As Long = -1, _

                                                 Optional FindAtStart As Boolean = False, _

                                                 Optional FindExact As Boolean = True) _

                                                      As Long

Argumente

sArray

Array, das nach der Phrade durchsucht werden soll

Find

Phrase, die in dem Array gesucht werden soll

StartIndex

Optional

Gibt die Untergrenze an, von wo an gesucht werden soll

EndIndex

Optional

Gibt die Obergrenze an, bis zu der gesucht werden soll

FindAtStart

Optional

Wird True übergeben, so muß die Phrase am Anfang des Feldes stehen

FindExact

Optional

Wird True übergeben, so wird zwischen Groß und Kleinschreibung unterschieden

Rückgabewert

Index der gefundenen Phrase

-1 wenn Phrase nicht gefunden wurde

-2 bei internen Fehler

 

Beispiel

    Dim A As Variant

    Dim Arr As New WindowAPI.Arry

   

    ReDim A(4)

   

    A(0) = "AAA"

    A(1) = "bbb"

    A(2) = "CcC"

    A(3) = "dDd"

    A(4) = "eeE"

   

    MsgBox Arr.FindPhrase(A, "eE", 1, 4, False, True)
