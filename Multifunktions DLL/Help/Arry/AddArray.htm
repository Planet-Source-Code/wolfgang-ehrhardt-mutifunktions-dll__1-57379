Function Arry.AddArray

Fügt ein Array an ein anderes Array hinzu

Deklaration

Public Function AddArray(sArray1 As Variant, _

                                               ByVal sArray2 As Variant) _

                                                           As Boolean

 

Argumente

sArray1

Array, dem sArray2 hinzugefügt werden soll

sArray2

Array, das sArray1 hinzugefügt werden soll

Rückgabewert

True wenn sArray1 erfolgreich sArray2 hinzugefügt wurde

 

Beispiel

    Dim Arr As New Arry

    Dim A() As String, B() As String

   

    ReDim A(2)

    ReDim B(2)

   

    A(0) = "0"

    A(1) = "1"

    A(2) = "2"

   

    B(0) = "3"

    B(1) = "4"

    B(2) = "5"

   

    MsgBox Arr.AddArray(A, B)
