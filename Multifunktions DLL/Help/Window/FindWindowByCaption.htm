Function Window.FindWindowByCaption

Findet ein Fenster anhand seines Textes in der Titelleiste

Deklaration

Public Function FindWindowByCaption(ByVal Caption As String) _

Argumente

Caption

Text in der Titelleiste das gefunden werden soll

Rückgabewert

Hwnd des gefundenen Fensters oder 0 wenn Fenster nicht gefunden wurde

 

Beispiel

'1 x Form

 

Private Sub Form_Load()

    Dim WIN As New WindowAPI.Window

   

    Me.Caption = "MeinProjekt"

   

    MsgBox WIN.FindWindowByCaption("MeinProjekt")

   

End Sub
