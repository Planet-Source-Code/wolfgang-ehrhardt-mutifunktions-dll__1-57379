Function Controls.CheckBox_AsOptionButton

Läßt eine Gruppe von CheckBoxen so wie Option-Buttons funktionierem

Deklaration

Public Function CheckBox_AsOptionButton(CheckBox As Object, _

                                                                             ByVal Index As Long) _

                                                                                  As Boolean

Argumente

CheckBox

Gruppe von CheckBoxen

Index

Der Index, der beim Click-Ereignis ausgelöst wird

(siehe Beispiel)

Rückgabewert

True wenn Operation erfolgreich

 

Beispiel

'1x Form

'4x CheckBox 'Check1 (Mit Index 0 bis 3)

 

Private Sub Check1_Click(Index As Integer)

    Dim CRTL As New WindowAPI.Controls

   

    Call CRTL.CheckBox_AsOptionButton(Check1, Index)

   

End Sub
