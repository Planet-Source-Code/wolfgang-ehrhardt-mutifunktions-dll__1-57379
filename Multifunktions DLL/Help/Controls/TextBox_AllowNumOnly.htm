Function Controls.TextBox_AllowNumOnly

Erlaubt nur eine numerische Eingabe in eine TextBox

Deklaration

Public Function TextBox_AllowNumOnly(TextBox As Object, _

                                                                 tbEvent As tbEVENTnfo, _

                                                                 Optional DigitsAfterDecimal As Long = 0, _

                                                                 Optional KeyCode As Integer, _

                                                                 Optional HelpOnF1 As Boolean = True, _

                                                                 Optional SetToolTip As Boolean = True) _

                                                                      As Boolean

Argumente

TextBox

TextBox, in der nur eine numerische Eingabe möglich sein soll

tbEvent

Optional

tbKEYDOWN

    Die Funktion wurde aus dem Key-Down-Ereigniss

    aufgerufen

tbCHANGE

    Die Funktion wurde aus dem Change-Ereigniss

    Aufgerufen

DigitsAfterDecimal

Optional

Bestimmt, wie viele Stellen hinter dem Komma eingegeben werden dürfen

KeyCode

Optional

Muß nur übergeben werden, wenn die Funtion aus dem Key-Down-Ereigniss aufgerufen wurde

HelpOnF1

Optional

Wird True angegeben un der User drückt in der TextBox F1, so erscheint ein Hinweisdialog mit den möglichen Tasten

SetToolTip

Optional

Wird True angegeben, so erhält die TextBox einen Tooltip mit Hinweis

Rückgabewert

True wenn der Ereignis richtig verarbeitet wurde

 

Beispiel

'1x Form

'1x TextBox 'Text1

 

Dim Crtl As New WindowAPI.Controls

 

Private Sub Form_Load()

    Text1.Text = ""

End Sub

 

Private Sub Text1_Change()

    Call Crtl.TextBox_AllowNumOnly(Text1, tbCHANGE, 2, , True, True)

End Sub

Private Sub Text1_KeyPress(KeyAscii As Integer)

    Call Crtl.TextBox_AllowNumOnly(Text1, tbKEYDOWN, 2, KeyAscii, True, _

                                                                 True)

End Sub
